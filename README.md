# 🧬 Project Human body

Project Human is a long-term scientific learning project focused on understanding the human body through biology, programming, and mathematical thinking.

The goal of this project is not memorization, but deep understanding of how biological systems work and how they can be modeled, analyzed, and explained using computational thinking.

---

## 🎯 Project Purpose

- Understand how human body systems function 
- Develop scientific thinking through questions, exploration, and analysis
- Connect biology with programming (Python) and logic
- Build structured learning and research habits

---

## 📁 Project Structure

- 📒 **Journal/**
  Daily learning notes, reflections, and progress tracking.

- 🧠 **Ideas.md**
  Scientific questions, hypotheses , and curiosity-driven thoughts.

- 🗺️ **Roadmap.md**
  The structured learning path and progression of the project.

- 📖 **README.md**
  Overview and explanation of the project.

---
## 🫀 Current Direction

Project Human does not focus on a single organ or system.

Instead, it explores the human body as an interconnected system, where biological processes (such as circulation, respiration, and neural activity) are studied in relation to each other.

The goal is to understand how different systems interact to maintain the function of the human body as a whole.

---

## 💻 Tools & Methods

- Python for simple biological modeling
- Self-learning through questions and exploration
- Structured documentation of ideas and progress

---

## 🚀 Long-term Vision

To develop a deep understanding of biological systems and the ability to think like a researcher by combining science, logic, and programming.
